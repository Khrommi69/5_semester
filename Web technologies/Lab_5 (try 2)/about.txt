SASS - Syntactically Awesome StyleSheets
SCSS - препроцессор основанный на CSS
мощный и гибкий инструмент формирования стилей Web-сайтов/приложений

SASS - нет {} и ;     SCSS - есть

SCSS
ПЕРЕМЕННЫЕ
$some-color: #fce123;

УСЛОВИЯ
@if (<условие>) {
 // команды выполняются, если <условие> истинно
}
@else {
 // команды выполняются, если <условие> ложно
}

ЦИКЛЫ
@for <переменная> from <старт> through <стоп> {
 // тело цикла
}

@while <условие>{
 // тело цикла с изменением шага
}

позволяет для каждого значения из определенного списка применить заданный набор команд
@each <переменная> in <значение1>, < значение2>... {
 // тело цикла
}
Переменная последовательно принимает значения из списка после
ключевого слова in. Чтобы воспользоваться ее текущим значением, можно
использовать конструкцию #{<переменная>}.

ВЛОЖЕНИЯ СЕЛЕКТОРОВ
.title {
 strong {}
 em {}
}

в случае с псевдоклассами/псевдоэлементами
// SCSS
.parent {                      CSS
 &:hover {}               ==> .parent:hover {}
}

ПРИМЕСИ (аналог функций)
@import @mixin @include

@mixin border-radius($radius: 1px) {
 -webkit-border-radius: $radius;
 -moz-border-radius: $radius;
 -ms-border-radius: $radius;
 border-radius: $radius;
}
.box{
 @include border-radius();
}
