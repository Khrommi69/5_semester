***CMD***
java "имя класса"
Error: Could not find or load main class "имя класса"
РЕШЕНИЕ ПРОБЛЕМЫ:
java -cp . "Имя класса"

"-cp .": это опция, которая устанавливает путь к классам, которые нужны для запуска программы.
Здесь "." означает текущую директорию. Это означает, что все необходимые классы для запуска "имя класса" должны находиться в текущей директории.
